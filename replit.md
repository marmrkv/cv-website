# Personal Portfolio - Margarita Markova

## Overview
A personal one-page portfolio website for a data analyst and dashboard creator. Features a dark, minimal design with smooth scroll animations.

## Project Architecture
- **server.js** - Express.js server serving static files on port 5000
- **public/** - Static assets directory
  - **index.html** - Main HTML page with all sections
  - **styles.css** - CSS styling (dark theme, responsive)
  - **script.js** - Scroll animations using Intersection Observer

## Tech Stack
- Node.js 20
- Express.js
- Vanilla HTML/CSS/JavaScript
- Google Fonts (Inter)

## Running the Project
The application runs on port 5000 with the command:
```
node server.js
```

## Sections
1. Hero - Name, title, positioning statement, LinkedIn CTA
2. What I Do - Value proposition
3. Experience - Work history highlights
4. Strengths & Skills - Grouped capabilities
5. Education & Credentials - Degrees and certifications
6. Contact - LinkedIn CTA

## User Preferences
- Dark & minimal visual style
- Clean, modern typography
- Single-column vertical scroll layout
